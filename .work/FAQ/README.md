# Project FAQ

This directory contains frequently asked questions, common pitfalls, and important architectural decisions.

## Structure

- **`pitfalls/`**: Common mistakes and how to avoid them
- **`decisions/`**: Architectural decisions and their rationale

## Purpose

- Prevent recurring mistakes
- Document why certain approaches were chosen
- Share knowledge across the team
- Guide future implementation decisions

## Contributing

When you encounter a pitfall or make an important decision, document it in the appropriate directory.
