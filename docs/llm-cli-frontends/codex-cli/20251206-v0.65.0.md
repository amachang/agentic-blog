# Codex CLI v0.65.0

Released: 2025-12-04

## Changes

- Codex Max is now the default model
- Fixed TUI panic issues related to async-in-sync code
- Added `/resume` slash command with performance enhancements
- Tips and tooltips now render via markdown
- Added Ctrl-P/N navigation
- Screen-line-capped shell output
- Windows clipboard image paste support
- History file trimming
- Exclusion of common directories like `__pycache__`
- Improved paste placeholder handling

## Development Activity

Alpha releases (0.66.0-alpha.1 through alpha.9) published December 4-6, indicating active development.
