# Nex-N1 調査メモ - 2025-12-08

> 論文: [Nex-N1: Agentic Models Trained via a Unified Ecosystem for Large-Scale Environment Construction](https://arxiv.org/abs/2512.04987)

## 概要

LLMを「受動的な応答者」から「自律的なエージェント」に進化させるためのpost-trainingフレームワーク。

## アーキテクチャ

**Post-training方式**: 既存のベースモデルにエージェント特化の学習を追加

```
ベースモデル (Qwen3, DeepSeek等)
        ↓
   Nex Post-training (合成データ + RL)
        ↓
   Nex-N1 シリーズ
```

### 公開モデル

| モデル | ベース | サイズ | リンク |
|--------|--------|--------|--------|
| Qwen3-32B-Nex-N1 | Qwen3-32B | 32B | [HuggingFace](https://huggingface.co/nex-agi/Qwen3-32B-Nex-N1) |
| DeepSeek-V3.1-Nex-N1 | DeepSeek-V3.1 | 671B | フラッグシップ |
| DeepSeek-V3.1-Nex-N1-GGUF | DeepSeek-V3.1 | 671B (量子化) | [HuggingFace](https://huggingface.co/mradermacher/DeepSeek-V3.1-Nex-N1-GGUF) |

8B〜671Bまでのサイズ展開あり。

## Nex エコシステム

3つのコンポーネントでインタラクティブ環境を大規模構築:

1. **NexAU**: 複雑なエージェント階層を簡単な設定で構築できるフレームワーク
2. **NexA4A**: 自然言語から多様なエージェント階層を自動生成（無限のドメインをカバー）
3. **NexGAP**: シミュレーションと現実のギャップを埋め、グラウンデッドな軌跡を合成

## ベンチマーク結果

- **SWE-bench**: オープンソースSOTAを上回る
- **tau2**: プロプライエタリモデルに匹敵する性能
